# Prompt Library

# Prompt Library

A library of Daloopa tested prompts, categorized by use case, and are great examples of our MCP's functionality for any investor profile.

## 

Easy Data Pulls

Querying which KPIs are available for which tickers, and returning values and source for analyses and data management. Tables formatted so that data can be directly copy and pasted into Excel for model building and maintenance purposes.

### 

Example Prompts

  * Retrieve all KPIs and values for inventory metrics related to CVNA for the past 4 quarters. Break out the metrics by quarters, and present the table with quarters as the column labels, and KPI names as the row headers. Provide me this data in table format in the chat and as a downloadable Excel file. Use Daloopa data and provide hyperlinks.

  * Retrieve all the KPIs I would need to perform a DCF for ATRC. Present the table with quarters as the column labels, and KPI names as the row headers. Use Daloopa data and provide hyperlinks.

  * Get stock buy-back information for the past 2 years for AAPL and META and break it out by quarters. Provide it in a table format with quarters as the column labels, and KPI names as the row headers. Use Daloopa data and provide hyperlinks.

  * Pull all data available highlighting major podcast deals that SPOT has conducted within the past 8 quarters. Use Daloopa data only and provide hyperlinks.

## 

Benchmarking

Comparing KPIs and performance across tickers within an industry or sector. Helpful for anyone building valuation comps, or trying to understand winners and losers within a sector.

### 

Example Prompts

  * Build a comparative dashboard for MSFT, ORCL, SAP, comparing cloud ARR growth, gross dollar retention and EBIT margin for the last 3 years. Break this out by quarters and show me both the data and growth rates calculated from the data. Use Daloopa data and provide hyperlinks. Provide me a downloadable excel with all the data in a table format.

  * Compare cash on cash returns for the last 4 quarters on opening a CAVA, SHAK, and CMG store. Use Daloopa data and provide hyperlinks.

  * Compare GLP1 sale trends for the past year across all major pharmaceutical companies. Use Daloopa data and provide hyperlinks.

  * Perform a dupont analysis for AMD and Intel. Use Daloopa data and provide hyperlinks (also falls under the Financial Analysis use case).

## 

Inflection Tracking

Identifying major shifts in KPIs or trends for positioning and thesis validation.

### 

Example Prompts

  * For [insert ticker here], highlight key Q/Q changes from the most recent financial data compared to the previous quarter. Walk me through the main accels and decels and margin changes. Use Daloopa data and provide hyperlinks.

## 

Financial Analysis

For users who want to perform complex financial analyses, with data synthesized directly in the analysis.

### 

Example Prompts

  * Run me a sensitivity analysis on ARPU and impacts on fiscal year 26 profit margins for NFLX. Use only Non-GAAP data, use Daloopa Data and provide hyperlinks.

  * Perform an efficiency analysis for Nike using the last 2 quarters of data. Break out the asset turnover ratio, inventory turnover ratio, and receivables turnover ratio. Use Daloopa data and provide hyperlinks.

  * Perform a dupont analysis for AMD and Intel. Use Daloopa data and provide hyperlinks (also falls under the Benchmarking use case).

## 

Research and Due Diligence

Helps users become a subject matter expert in no time on a name where they're building expertise. Helpful for those building coverage, or trying to understand trends for priority names in coverage or watchlist.

### 

Example Prompts

  * I am initiating coverage on US mining industry coverage - generate me an industry report summarizing key trends across the last year, highlighting largest over and underperforming tickers, breaking out key KPIs that drove over or under performance. Use Daloopa data and provide hyperlinks.

  * What was CVNA's financing attach rate for the past 8 quarters? Use Daloopa and provide hyperlinks.
