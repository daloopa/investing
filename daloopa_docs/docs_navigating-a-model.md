# Navigating a model

## 

Key Sections

Each Excel model will have the following sections:

  * Dates
  * Documents
  * Guidance
  * KPIs
  * Segmental breakdown
  * Geographical breakdown
  * Income statement
  * Balance sheet
  * Cash flow statement
  * GAAP to Non-GAAP bridges
  * Others

## 

Sources

Data is collected from publicly available company filings, including:  
10-K / 10-Q / Annual reports / Quarterly reports  
8-K (Press release / earnings release)  
Investor presentations

  


## 

Data Collection Philosophy

We collect data as-is from company-reported filings. We do not edit or alter in any way the data the company provides. For instance, if the company provides a detailed cash flow statement breakdown, we will collect every single item and present them in our Excel models.

All models are checked multiple times to ensure that not only is the data accurate, but also the data relationships are accurate.
